# Codex Start Prompt

Lavora su Blue Airco Support rispettando integralmente `AGENTS.md` e la documentazione in `docs/`.

Prima di modificare il progetto:

1. verifica specifica, architettura e matrice di tracciabilità;
2. dichiara assunzioni e informazioni mancanti;
3. non inventare dati tecnici o fonti;
4. mantieni obbligatoria l'approvazione umana;
5. aggiungi test proporzionati alle modifiche ed esegui la suite;
6. non integrare OpenAI, WhatsApp o risposte automatiche senza richiesta esplicita.
